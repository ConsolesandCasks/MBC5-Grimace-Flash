## BOM
| Ref | Part |
| ------------- | ------------- |
| Resistors  | 10k 0603  |
| Capacitors  | 0.1uF 0603  |
| FRAM | FM18W08 |
| Flash | AM29F016B (2MB) or AM29F032/MBM29F033 (4MB) |
| U4 | 74LVC1G332 |
| MBC5 | Harvested from OEM Cart |
